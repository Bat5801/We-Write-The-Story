========================================
  三体 DLC - 安装说明
========================================

欢迎来到三体世界！这是一个基于《三体》小说的粉丝作品，
包含了原著三本小说的剧情，并增加了一些有趣的IF线。

========================================
  安装方法
========================================

方法一：批量导入（推荐）
1. 打开 We Write The Story 游戏
2. 点击"开始游戏"进入选关界面
3. 点击"批量导入故事"按钮
4. 在弹出的文件选择窗口中，按住 Ctrl 键（或 Shift 键）
   选择本文件夹中的所有 .story 文件：
   - 3body_part1.story（三体·第一部）
   - 3body_part2.story（三体·第二部）
   - 3body_part3.story（三体·第三部）
5. 点击"打开"完成导入

方法二：复制到 stories 文件夹
1. 将本文件夹中的所有 .story 文件复制到游戏目录的 stories/ 文件夹中
2. 运行游戏即可在选关界面看到三体系列

========================================
  故事介绍
========================================

📖 三体·第一部 - 地球往事
   跟随叶文洁的视角，见证人类文明与三体文明的初次接触。
   你将面临艰难的选择：是否发送信号？是否回复三体文明？

📖 三体·第二部 - 黑暗森林
   作为面壁者罗辑，你需要制定对抗三体文明的策略。
   黑暗森林法则是人类唯一的希望吗？

📖 三体·第三部 - 死神永生
   作为新执剑人程心，你掌握着两个文明的命运。
   是维持威慑，还是寻求和平？

========================================
  特色功能
========================================

✅ 多个IF线：每个故事都有多个分支和结局
✅ 物品系统：收集关键物品，影响剧情走向
✅ 地图系统：探索不同的场景和地点
✅ 多种结局：你的选择将决定人类文明的命运
